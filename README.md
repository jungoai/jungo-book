# JungoAI Book

This repository contains the source of "JungoAI" book.

## Requirements

Building the book requires [mdBook](https://rust-lang.github.io/mdBook/index.html).

```
cargo install mdbook
```

## Building

To build the book:

```
mdbook build
```

## Run Locally

To serve web page locally:

```
mdbook serve
```
